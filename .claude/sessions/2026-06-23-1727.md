## Decisions Made
- **No `dotenv` dependency** — used Node's built-in `process.loadEnvFile()` (guarded by a
  `typeof` check for older runtimes, skipped in production where env is injected). Keeps the
  dependency surface minimal.
- **JWT secrets have no defaults and require ≥32 chars** — enforces CLAUDE.md "no secrets in
  code"; missing secrets must fail at boot rather than silently using a weak fallback.
- **TTLs kept as duration strings** (`15m`/`7d`), validated by regex, so they can be handed
  straight to the JWT lib in the Phase 1 auth module without re-parsing.
- **`config` object is frozen** (`Object.freeze`) to prevent accidental mutation at runtime.

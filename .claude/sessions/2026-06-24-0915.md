## Decisions Made
- **Deferred helmet/cors/rate-limit to Phase 4** (not registered now) — the roadmap scopes them to
  the "Security pass" + "Rate limiting" Phase 4 items; registering here would hollow those out.
  Deps are already installed and ready. Phase 0's base-plugins line is error handler + swagger.
- **Harness test needs no DB** — the first HTTP test mounts only the error-handler plugin on a bare
  Fastify app (not the full `buildApp`, which eagerly `$connect()`s Prisma). Keeps `npm test` green
  from a clean checkout and keeps CI free of a DB service for now; real DB-backed integration tests
  come with Phase 1 (CI will gain a postgres service then).
- **Error handler trusts `error.statusCode < 500`** to pass through framework errors (rate-limit
  429, body-parse 400s) rather than masking them as 500s.
